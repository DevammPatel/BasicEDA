# 📊 Netflix Data Analysis & Feature Engineering
This project focuses on Exploratory Data Analysis (EDA) and Feature Engineering (FE) on a Netflix dataset to uncover insights about content distribution, trends, and patterns.

## 📁 Dataset
The dataset used in this project is sourced from Kaggle:
🔗 [Netflix Data Cleaning, Analysis and Visualization](https://www.kaggle.com/datasets/ariyoomotade/netflix-data-cleaning-analysis-and-visualization)

It contains information about Netflix shows including:
- Title
- Director
- Cast
- Country
- Date added
- Release year
- Rating
- Duration
- Genre (Listed_in)

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Engineer meaningful features for better insights
- Visualize trends and patterns in Netflix content

## 🧹 Data Cleaning

Key steps performed:
- Handling missing values
- Converting data types (e.g., dates)
- Removing duplicates
- Standardizing categorical values

## 📊 Exploratory Data Analysis (EDA)

Some of the insights explored:
- Distribution of Movies vs TV Shows
- Content growth over the years
- Most common genres
- Country-wise content production
- Ratings distribution

## ⚙️ Feature Engineering

Created new features to enhance analysis:
- year_added from date_added
- month_added for seasonality trends
- Content duration categories
- Genre splitting for multi-label analysis

## 📈 Visualizations
- Bar charts for categorical distributions
- Time-series plots for growth trends
- Heatmaps and count plots for deeper insights

## 🛠️ Tech Stack
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
